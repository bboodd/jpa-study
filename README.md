## jpa, postgresql, maven 프로젝트

## Reference: https://github.com/spring-projects/spring-petclinic

### 실행방법

```bash
git clone https://github.com/bboodd/jpa-study.git
cd jpa-study
./mvnw package
java -jar target/*.jar
```
